Support Vector Machine

前言

SVM是一种非常重要的机器学习分类算法，在DeepLearning普及之前占据着分类算法主导地位，放在今天仍然有很多场景会用到SVM，这篇文章将详细讲解SVM，文末会有Matlab的SVM实现源码，方便读者理解原理。

SVM简介

SVM是一种有监督的机器学习算法，用来对两类数据进行分类。

整个SVM算法有以下几个重点：

  • Margins
  • Primal
  • Lagrange
  • Dual
  • Soft Margin
  • Kernel

Margins

Margin，也就是间隔，是SVM理论中非常关键的理念。

Margin 总的来说有三类：

  • Functional Margin
  • Geometric Margin
  • Margin of a training set

我们可以再细分：

  • $\gamma_{i}^{f}$ : Functional margin of an example $\mathbf({x_i, d_i})$
  • $\gamma_{i}^{g}$: Geometric margin of an example $\mathbf({x_i, d_i})$
  • $\gamma^{f}$: Functional margin of a training set
  • $\gamma^{g}$: Geometric margin of a training set
  • $\gamma$: Margin of a training set
